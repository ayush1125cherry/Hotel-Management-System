# Hotel Management System

A full-stack Hotel Management System inspired by modern booking platforms like Booking.com and Airbnb.

This project allows users to:
- Browse hotels
- Search rooms
- Manage bookings
- Authenticate securely using JWT
- Experience a responsive modern UI
